# JDK环境变量配置

## 1.0 CLASSPATH配置

```java
CLASSPATH
.;%JAVA_HOME%\lib;%JAVA_HOME%\lib\dt.jar;%JAVA_HOME%\lib\tools.jar
```
<font color=#ff4d4d>**注意:不要遗漏前面的`.;`**</font>
